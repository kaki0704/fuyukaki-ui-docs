# fuyukaki-ui Documentation

Official documentation site for fuyukaki-ui - A modern, accessible React UI component library with Warm & Organic design philosophy.

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```

## Tech Stack

- TanStack Start
- React 18
- TypeScript
- fuyukaki-ui
